CSS là gì?

CSS là viết tắt của Cascading Style Sheets, nó là một ngôn ngữ được sử dụng để tìm và định dạng các phần từ được tạo bởi các ngôn ngữ đánh dấu(HTML).

Với CSS, bạn có thể áp dụng nhiều kiểu dạng khác nhau cho cùng một trang HTML. Nhờ đó, lập trình viên cs thể tạo ra nhiều phiên bản khác nhai cho một trang web chỉ định bằng cách điều chỉnh các quy tắc CSS

Tác Dụng của CSS
1. Giải quyết vấn đề lớn
2. Tiết kiệm rất nhiều thời gian
3. Cung cấp thêm các thuộc tính
